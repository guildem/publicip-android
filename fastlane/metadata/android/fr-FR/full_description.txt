Application et Widget permettant à l'utilisateur de trouver son adresse IP publique.

Cette application utilise l'API ipify pour trouver l'IP.
Les sources de ce service peuvent être trouvées ici : https://github.com/rdegges/ipify-api.
Il n'enregistre aucune donnée. Voir ici pour plus de détails : https://www.ipify.org.
